一款主打安全的即时通讯APP：Telegram

Jul 28, 2017 | 杂谈 | 阅读
0 条评论

Telegram是我偶然间从某群中看到的，因为其超高的安全性，所以被很多国家强力封锁，中国当然也不例外，所以如果想用得需要自备梯子到PlayStore下载。
telegram的优点是速度快，图片质量高，即时通讯做的是一大亮点，比如你在电脑上的Telegram输了一些文字，然后在手机上打开Telegram，你就会看到你在电脑上输了什么东西在手机上就会有什么，而且光标位置也相同！
刚开始用可能是云里雾里的，不过有一晚上就能发现它的强大快捷，而且谈论的内容层次都不一样。
