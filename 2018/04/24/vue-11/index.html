vue 元素中使用 key 值

Apr 24, 2018 | 前端 | 阅读
0 条评论

vue 有一个机制就是 尽可能的复用当前页面上的 DOM 节点 ，但是在使用 v-if/else 的时候会发现 当切换元素的时候, vue 并不会清空之前输入的值，因为 vue 会认为他是一个可以重用的元素，所以，要解决这种情况可以使用 key 来声明页面的唯一元素

key 在循环列表中使用的情况也有很多，在列表循环的时候，我们需要每条数据都有一个唯一的主键，这个主键不能用 循环的来的下标，也就是不能用 index ，我们要用的是后端传过来的 id 值
