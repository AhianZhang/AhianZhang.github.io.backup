配置

配置环境变量，同 Windows 下一样目的是可以在非命令所在目录下使用。

不管是 CentOS 还是 Ubuntu 在 /etc 下都有一个 profile 文件，此文件就是我们需要配置的环境变量文件，Java 的环境变量也是在这里的，配置完后可以在任意地方任意用户下使用命令。
sudo vi /etc/profile

注意：在 Linux 系统中环境变量使用冒号 :分隔，而非分号’;’

修改完成后刷新配置 source /etc/profile

测试

配置完环境变量后使用 which hadoop 命令进行测试，如果回显出位置，则配置成功。
