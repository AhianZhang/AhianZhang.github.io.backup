微信小程序简介及环境配置

Mar 4, 2018 | 杂谈 | 阅读
0 条评论

微信小程序是一种全新的连接用户与服务的方式，它可以在微信内被便捷地获取和传播，同时具有出色的使用体验。

有关怎么注册以及其怎么开发在官网上已经写的很详细了，本文主要讲微信小程序的优势和注意事项以及建议。

优势

“小程序” 顾名思义，它很小，但功能很完善，能将绝大部分的后端（如 Java Python NodeJs 等）数据渲染到小程序中；它的语法很简单，在开发手册中已经全部罗列出来；用户量越大，流量也就越大，微信在国内的聊天 APP 上排在第一，用户量可想而知，在近一年里微信小程序如雨后春笋般的冒了出来，而且用户也很欣然接受了这样的方式（开发的一切应用不都是从用户角度出发嘛）；能够迅速的进行迭代。

注意事项

小程序只是用来实现本地化 APP 的部分功能，可以用它来开发一些简单的低成本的项目，比如抽奖、宣传推广、小店点餐等，对于庞大的电商系统及涉及资金的项目最好避免使用小程序。

小程序请求访问的路径必须是 https 开头的，在开发模式可以为 http 但实际上线测试时无法获取资源。

建议

玩玩就好，业余时间练练，并不推荐靠开发小程序挣钱（单纯的技术路线）。
